Konidaris' Pinball Domain now Available in Python

As part of my Master’s thesis, I implemented George Konidaris’ Pinball domain for reinforcement Learning. Konidaris had already made available his original implementation in Java. The environment could also be reached through an RL-Glue socket interface.

Porting it to Python however made the transition to Will Dabney’s Python-Rl much easier while avoiding the performance bottleneck due socket communication.

And I love working with Python
