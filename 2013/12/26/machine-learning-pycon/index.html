Machine Learning and Data Crunching at Pycon

26 December 2013

Pycon is taking place in Montreal next year from April 9 to 17. Here are a few talks related to machine learning and data mining:

  • How to Get Started with Machine Learning by Melanie Warwick @nyghtowl
  • Know Thy Neighbor: Scikit and the K-Nearest Neighbor Algorithm by Portia Burton @pkafei
  • Realtime predictive analytics using scikit-learn & RabbitMQ by Michael Becker
  • Enough Machine Learning to Make Hacker News Readable Again by Ned Jackson Lovely @nedjl
  • Diving into Open Data with IPython Notebook & Pandas by Julia Evans @b0rk
  • Data intensive biology in the cloud: instrumenting ALL the things by Titus Brown

The conference talk list can be found here: https://us.pycon.org/2014/schedule/talks/list/

Tutorials

  • How to formulate a (science) problem and analyze it using Python code
  • Dynamics and Control with Python
  • Diving deeper into Machine Learning with Scikit-learn by Olivier Grisel @ogrisel
  • Hands-on with Pydata: how to build a minimal recommendation engine.
  • Bayesian statistics made simple
  • Mining Social Web APIs with IPython Notebook
  • Exploring Machine Learning with Scikit-learn by Jake Vanderplas @jakevdp and Olivier Grisel

Unfortunately, according to https://us.pycon.org/2014/registration/#Tutorial it seems that each tutorial costs $200. A price which makes sense for companies and self-employed contractors, but not so much for students or general enthusiasts.

Financial aid is offered for the main conference or on a per-tutorial basis: https://us.pycon.org/2014/assistance/
