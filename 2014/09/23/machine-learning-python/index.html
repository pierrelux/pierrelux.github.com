An intro to applied machine learning in Python

Sep 23, 2014

I wrote this IPython notebook on using Python for applied machine learning. The material was intended for grad students taking the course Applied Machine Learning taught by professor Joelle Pineau.

I tried to cover some of the language interoperability features of IPython and then focused on model selection and preprocessing under Scikit-learn.

You can view and download the notebook on Nbviewer
